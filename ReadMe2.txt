Ps: if you are using it on Linux, consider to run the commands to configure HTTPS endpoint:
"dotnet dev-certs https --clean";
"dotnet dev-certs https -t";
"dotnet dev-certs https --check".

(Do not copy the quotation marks, semicolon and the dot).
